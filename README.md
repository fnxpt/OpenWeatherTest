# OpenWeatherMap Test

## Instructions

1. Install brew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

2. Install brew bundle
```
brew tap homebrew/bundle
```


3. Install
```
brew bundle
```

4. Add key on API.swift
